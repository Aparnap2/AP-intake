Invoice Upload Demo

Quick Test Instructions

  1. Start the development server: npm run dev
  2. Start the backend API: docker-compose up -d
  3. Navigate to http://localhost:3000/invoices
  4. Click the "Upload Invoice" button
  5. Test with PDF or image files
  6. Check browser console for API responses

Test Files

You can test the upload functionality with:

  • PDF files (preferred)
  • PNG images
  • JPG/JPEG images
  • TIFF images

File size limit: 50MB
Maximum files: 10 per upload

Expected Behavior

Modal Opens: Upload modal should appear when clicking the upload button
Drag & Drop: You can drag files onto the drop area or click to browse
File Validation: Files are validated for type and size
Progress Tracking: Upload progress is shown for each file
Error Handling: Failed uploads show error messages
List Refresh: Invoice list refreshes after successful upload

API Endpoints

POST /api/v1/invoices/upload

Uploads invoice files for processing

GET /api/v1/invoices

Retrieves list of processed invoices

GET /api/v1/invoices/{id}

Retrieves specific invoice details

Testing with Playwright

Run the automated tests:

# Install dependencies
npm install
# Run tests
npx playwright test tests/invoice-upload.spec.ts
# Run tests in headed mode
npx playwright test tests/invoice-upload.spec.ts --headed
